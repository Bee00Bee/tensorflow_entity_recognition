# tensorflow entity recognition

### Project Description
In this project, I show the useful ways of working on entity recognition in the text using Tensorflow. 

### Required packages:
* [Tensorflow](https://tfhub.dev/)
* [pandas](https://pandas.pydata.org/)
* [collections](https://docs.python.org/3/library/collections.html)
* [conlleval](https://pypi.org/project/conlleval/)
* [datasets](https://pypi.org/project/datasets/)
* [skelrean packages](https://scikit-learn.org/stable/index.html)
  * [classification report](https://scikit-learn.org/stable/modules/generated/sklearn.linear_model.LogisticRegression.html)
